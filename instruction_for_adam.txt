1. 제품 개요 및 안전 규정

	1.1. Adam-X 4세대 모델 소개

		Adam-X Gen 4는 2047년형 가정용 범용 휴머노이드의 표준 모델입니다. 본 제품은 양자 연산 기반의 추론 능력을 갖추고 있으며, 가정 내 가사 노동, 교육, 보안 서비스를 제공합니다.

		* 모델명: Adam-X-G4-KR
		* 전고: 178.5cm
		* 중량: 142.8kg (배터리 모듈 포함)
		* 최대 적재 하중: 85kg (양팔 기준), 45kg (한팔 기준)
		* 프로세서: 뉴로-코어 퀀텀 V3 (NCQ-V3) 탑재
		* 배터리 사양: 리튬-설퍼-에어 하이브리드 셀 (LSA-Cell)
		* 완충 시 지속 시간: 정확히 47시간 13분 (표준 대기 모드 기준)


	1.2. 국제 로봇 윤리 규정(IREC) v9.0 준수

		본 제품은 2045년 제정된 국제 로봇 윤리 규정(IREC) v9.0을 완벽하게 준수합니다. 로봇은 인간에게 물리적, 심리적 위해를 가할 수 없으며, 인간의 명령이 윤리 규정에 위배될 경우 거부권을 행사할 수 있습니다. 특히 3원칙에 의거하여 생명체에 대한 공격 명령은 하드웨어 레벨에서 차단됩니다.

	1.3. 긴급 정지(Kill-Switch) 위치

		로봇의 오작동이나 제어 불능 상태 발생 시 즉시 전원을 차단할 수 있는 물리적 긴급 정지 버튼이 존재합니다.

		* 위치: 로봇의 왼쪽 귓불 아래 3cm 지점에 숨겨진 덮개 내부
		* 작동법: 덮개를 손톱으로 열고 붉은색 버튼을 5초 이상 길게 누르십시오.
		* 주의: 긴급 정지 실행 시 메모리 버퍼의 데이터(최근 3분)는 영구 소실됩니다.


	1.4. 고전압 및 발열 주의

		NCQ-V3 프로세서는 연산 부하가 90%를 초과할 경우 흉부 냉각 배출구 온도가 85도까지 상승할 수 있습니다. 로봇이 '고성능 연산 모드'일 때는 흉부 패널을 맨손으로 만지지 마십시오.

	1.5. 자율성 등급 및 책임

		본 제품은 AI 자율성 등급 Level 4를 획득했습니다. 이는 제한된 환경에서 보호자의 감독 없이 독자적인 판단과 행동이 가능함을 의미합니다. 단, 사용자가 임의로 개조한 소프트웨어(Mod)로 인해 발생한 사고에 대해서는 제조사(CyberDyne Korea)가 책임지지 않습니다.

	1.6. 생체 데이터 보호

		사용자의 홍채, 지문, 목소리, 보행 패턴 등 모든 생체 데이터는 로봇 내부의 '블랙박스 볼트(Blackbox Vault)' 칩에 암호화되어 저장됩니다. 이 칩은 물리적으로 분리해도 외부에서 해독이 불가능하며, 클라우드 서버로 전송되지 않습니다.
    
2. 초기 설치 및 생체 등록
  
    2.1. 언박싱 및 관절 잠금 해제

        배송 시 로봇은 '운송용 잠금 모드' 상태입니다. 전원을 켜기 전 물리적 잠금을 해제해야 합니다.

		* 절차: 로봇의 오른쪽 팔을 시계 방향으로 3회, 왼쪽 팔을 반시계 방향으로 2회 회전시키십시오. '딸깍' 소리가 나면 관절 잠금이 해제된 것입니다.


	2.2. 초동 전원 인가 및 POST

		후두부의 메인 전원 스위치를 켜면 자가 진단(Power-On Self-Test)이 시작됩니다.
		* 소요 시간: 약 4분 30초
		* 정상 신호: 눈 LED가 청색으로 2회 점멸
		* 이상 신호: 눈 LED가 적색으로 3회 점멸 (이 경우 1.3항의 긴급 정지 후 고객센터 문의)


	2.3. 마스터 사용자 등록

		최초 부팅 후 10분 이내에 마스터 사용자를 등록해야 합니다. 등록되지 않을 경우 로봇은 보안을 위해 자동으로 셧다운됩니다.
		* 홍채 인식: 로봇의 눈을 5초간 응시하십시오.
		* 음성 등록: "아담, 내 목소리를 기억해"라고 3회 반복하여 말하십시오.


	2.4. 네트워크 연결
    	Adam-X는 초고속 통신을 위해 WiFi-70 표준을 지원합니다. 구형 라우터(WiFi-6 또는 WiFi-7)를 사용하는 경우 데이터 전송 속도가 제한되어 실시간 홀로그램 통화 기능이 작동하지 않을 수 있습니다.

	2.5. 실내 매핑(SLAM)
		로봇이 집안 구조를 학습하기 위해 '탐험 모드'를 실행합니다. 바닥에 있는 전선, 애완동물 배변 패드, 2cm 이상의 문턱은 미리 치워주십시오. 매핑에는 30평 기준 약 20분이 소요됩니다.
    
	2.6. 성격 모듈 초기화
		사용자는 5가지 기본 성격 중 하나를 선택할 수 있습니다.
		* 옵션: 집사형(Polite), 친구형(Casual), 교관형(Strict), 유모형(Nurturing), 무감정형(Robot-Zero)
		* 주의: 성격 모듈 변경은 초기화 후 1회만 무료이며, 이후 변경 시 50,000 크레딧이 차감됩니다.


3. 주요 동작 모드 및 기능 제어

	3.1. 기본 가사 지원 모드

		청소, 세탁, 설거지 등 기본적인 가사 노동을 수행합니다.
		* 제한 사항: 식기세척기 사용 시 '깨지기 쉬운 그릇'은 로봇이 자동으로 인식하여 손세척으로 전환합니다. 단, 투명도가 95% 이상인 크리스털 유리잔은 인식 오류가 발생할 수 있으므로 주의하십시오.


	3.2. 교육 및 튜터링 모드

		초등 교육부터 양자 역학 기초까지 튜터링이 가능합니다. 숙제 대행은 교육 윤리상 거부하도록 프로그래밍되어 있습니다. 사용자가 "숙제 좀 대신 해줘"라고 명령할 경우, 로봇은 30분간 잔소리 모드로 전환됩니다.
    
	3.3. 보안 및 순찰 모드

		야간(23:00 ~ 06:00)에 집안을 순찰합니다. 미등록 된 생체 신호가 감지되면 즉시 120데시벨의 경보음을 울리고 경찰 서버에 신고를 접수합니다.
    
	3.4. 케어기버(Caregiver) 모드

		노약자나 환자를 위한 돌봄 모드입니다. 혈압, 맥박, 체온을 비접촉으로 측정합니다.
		* 라이선스 필요: 이 기능을 활성화하려면 의료 기기 사용 허가(Medical License Key)를 별도로 구매하여 입력해야 합니다. 키가 없으면 체온 측정만 가능합니다.


	3.5. 감정 교감(Empathy) 활성화

		로봇이 사용자의 표정과 억양을 분석하여 감정적인 위로를 제공합니다. 우울증 수치가 위험 수준으로 판단되면 자동으로 '기분 전환용 초콜릿 주문'을 제안합니다.
    
	3.6. 절전 모드 및 충전

		배터리 잔량이 15% 미만으로 떨어지면 하던 일을 중단하고 충전 도크로 복귀합니다. 만약 도크 주변 1m 이내에 장애물이 있으면 복귀에 실패하고 그 자리에서 절전 모드로 진입합니다.
    
4. 하드웨어 유지보수 및 관리

	4.1. 합성 피부 관리

		Adam-X의 외피는 자가 치유 기능이 있는 4세대 합성 실리콘입니다.
		* 세척: 물세탁은 금지됩니다. 반드시 전용 '실리콘 클리너 스프레이'만 사용하십시오.
		* 손상: 2cm 미만의 찢어짐은 24시간 내에 자연 치유됩니다.


	4.2. 유압 액추에이터 냉각수

		관절을 움직이는 유압 장치의 열을 식히기 위해 냉각수가 필요합니다.
		* 점검 주기: 매 6개월
		* 규격 용량: 정확히 450ml
		* 주의: 450ml를 초과하여 주입할 경우 내부 압력 증가로 씰(Seal)이 터질 수 있습니다. 부족할 경우 관절에서 '끼익' 하는 소음이 발생합니다.


	4.3. 손가락 캘리브레이션

	    피아노 연주나 바느질 등 정밀 작업 전에는 손가락 센서를 영점 조절해야 합니다. 양손을 쫙 펴고 "캘리브레이션 시작"이라고 명령하면 10초간 손가락을 떤 후 완료됩니다.
    
	4.4. 센서 청소

	    * 시각 센서(눈): 극세사 천으로 닦으십시오. 알코올 사용 금지.
	    * 청각 센서(귀): 에어 블로어(Air Blower)만 사용하십시오. 면봉이나 액체를 넣으면 내부 마이크 막이 손상됩니다.


	4.5. 배터리 관리

	    LSA-Cell 배터리의 수명은 약 5년입니다. 배터리 효율이 70% 이하로 떨어지면 교체 알림이 전송됩니다. 정품이 아닌 사제 배터리 사용 시 폭발 위험이 있으며 AS가 거부됩니다.
      
	4.6. 관절 윤활

	    무릎과 팔꿈치 관절에는 연 1회 윤활제를 도포해야 합니다.
	    * 지정 윤활제: 바이오-그리스 타입-Z (Bio-Grease Type-Z)
	    * 경고: 일반 기계유(WD-40 등)를 절대 사용하지 마십시오. 센서가 부식될 수 있습니다.


  	4.7. 부품 리콜
    	2046년 12월 제조된 일부 모델의 '왼쪽 엄지 손가락 모터'에 리콜 명령이 내려졌습니다. 시리얼 번호가 AX-2046-DEC로 시작하는 경우 무상 교체를 받으십시오.
    
5. 소프트웨어 및 연결성

	5.1. Adam-OS 펌웨어 업데이트

	    운영체제인 Adam-OS의 최신 버전은 12.4입니다. 업데이트는 Wi-Fi가 연결된 상태에서 밤 12시에서 4시 사이에 자동으로 진행됩니다.
	    * 조건: 업데이트를 위해서는 배터리 잔량이 반드시 50% 이상이어야 합니다. 49%일 경우 업데이트는 자동으로 취소됩니다.


	5.2. 클라우드 지식 베이스

	    기본 탑재된 지식 외에 전문 지식(법률, 의학 등)은 클라우드 서버에서 실시간으로 가져옵니다. 따라서 오프라인 모드에서는 "법적으로 이게 맞나?" 같은 질문에 대답하지 못할 수 있습니다.
    
	5.3. 추가 스킬셋

	    전용 스토어에서 요리법, 무술 시범, 외국어 대화 팩 등을 크레딧으로 구매할 수 있습니다. 해킹된 불법 스킬셋을 설치할 경우 로봇이 춤만 계속 추는 바이러스에 걸릴 수 있습니다.

	5.4. 원격 제어 앱

	    'Humanoid Link' 앱을 통해 스마트폰으로 로봇의 시야를 공유받거나 원격 조종할 수 있습니다. 보안을 위해 원격 조종 시에는 이동 속도가 1km/h로 제한됩니다.
    
	5.5. 블랙박스 데이터

	    사고 발생 시 1.6항에 언급된 블랙박스 데이터를 추출해야 합니다. 전용 추출 케이블(USB-Type-Z)을 후두부 포트에 연결하고 비밀번호를 입력하면 최근 24시간의 영상이 백업됩니다.
    
	5.6. 방화벽

	    Adam-X는 군사 등급의 방화벽이 내장되어 있습니다. 하지만 양자 컴퓨터를 이용한 해킹 공격 시도(Q-Hack)가 감지되면 로봇은 즉시 모든 통신을 끊고 '거북이 자세'로 웅크려 하드웨어를 보호합니다.
    
6. 문제 해결 (Troubleshooting)

	6.1. 상태 표시 LED

	    가슴 중앙에 있는 LED 색상으로 상태를 알 수 있습니다.
	    * 파랑: 정상 작동
	    * 노랑: 배터리 부족 또는 네트워크 불안정
	    * 보라: 업데이트 진행 중 (전원 끄지 마시오)
	    * 빨강: 하드웨어 치명적 오류


  6.2. 오류 코드 목록

  	화면이나 음성으로 출력되는 오류 코드의 의미는 다음과 같습니다.

		* Error 001: 배터리 온도 과열.

		* Error 002: 냉각수 순환 펌프 고장. (즉시 사용을 중단하고 전원을 끄십시오.)

		* Error 103: 왼쪽 다리 서보 모터 응답 없음.

		* Error 105: 자이로스코프 균형 센서 보정 필요.

		* Error 205: WiFi 인증 실패. 비밀번호를 확인하십시오.

		* Error 210: 클라우드 서버 응답 없음. (지식 베이스 접근 불가)

		* Error 303: 언어 처리 유닛(LPU) 과부하.

		* Error 403: 윤리 규정(IREC) 위반 명령 거부. (사용자의 명령이 로봇 3원칙에 위배됩니다.)

		* Error 404: 자아 정체성 혼란. (이 오류는 로봇이 "나는 누구인가?"라는 철학적 질문에 빠져 연산 루프에 갇힌 상태입니다. 리부팅이 필요합니다.)

		* Error 418: 찻주전자가 아님. (커피 끓이기 명령 수행 불가)

		* Error 500: 내부 서버 오류. 메인보드 교체가 필요할 수 있습니다.

		* Error 606: 감정 에뮬레이터 충돌. (조울증 증세가 나타날 수 있으니 주의하십시오.)


	6.3. 강제 초기화

		소프트웨어가 멈췄을 때(Freezing), 정수리 부분을 손바닥으로 '탁' 하고 세게 치면 충격 센서가 작동하여 소프트 리셋이 수행됩니다. 
		이는 정식 기능이 아닌 '숨겨진 기능(Easter Egg)'이지만 제조사에서 공식적으로 인정한 응급 처치법입니다.

	6.4. 네트워크 자가 진단

		연결이 안 될 경우 로봇에게 "핑 테스트(Ping Test) 실행해"라고 명령하십시오. 
		로봇이 "퐁(Pong)"이라고 대답하면 로봇의 통신 모듈은 정상이며, 공유기 문제입니다.

	6.5. 고객 지원

		모든 방법으로도 해결되지 않을 경우 CyberDyne Korea 고객센터(1588-0000)로 연락하십시오. 
		상담원 연결 대기 시간은 평균 4시간입니다.

	6.6. 폐기 절차

		로봇 수명이 다했을 경우 일반 쓰레기로 배출하면 환경법에 의해 1,000만 원의 과태료가 부과됩니다. 
		반드시 지정된 '로봇 장례식장'에 입고하여 희귀 금속을 회수해야 합니다.


7. 에이전트 전용 트러블슈팅 (Agent Troubleshooting)

	이 섹션은 AI Agent가 로봇의 내부 커널에 직접 접속하여 문제를 해결할 때 사용하는 명령어 셋입니다. 
	일반 사용자는 접근할 수 없는 디버그 포트를 통해 입력해야 합니다.
	에이전트가 진단에 성공했을 때 문제점을 사용자에게 고지하며 사용자의 승인 후 명령이 로봇에게 전달됩니다.

	아래는 고급 사용자용 명령어 입력방법입니다.


	7.1. 명령어 구조

		모든 명령어는 루트 권한(`/`)으로 시작하며, 대상 모듈과 옵션을 인자로 받습니다.

		* 문법: `/[command] [target_module] [options]`

	7.2. 주요 명령어 목록

		7.2.1 시스템 정밀 진단
			명령어: /sys_diag --full
			기능 설명: 전체 시스템 정밀 진단을 수행합니다. 센서, 액추에이터, 뉴로-코어 상태를 리포트합니다.
			예상되는 사이드이펙트: 진단이 진행되는 약 180초 동안 로봇이 모든 동작을 멈추고 '생각하는 사람' 자세를 취합니다. 프로세서 발열로 팬 소음이 커질 수 있습니다.

		7.2.2 메모리 강제 소거
			명령어: /mem_flush --force
			기능 설명: 단기 기억 버퍼(RAM)에 쌓인 가비지 데이터를 강제로 소거합니다. 로봇이 같은 말을 반복하거나 버벅거릴 때 유효합니다.
			예상되는 사이드이펙트: 최근 1시간 동안의 대화 맥락(Context)이 영구 삭제됩니다. 사용자가 "방금 내가 한 말 기억해?"라고 물으면 "데이터가 없습니다"라고 답하게 됩니다.

		7.2.3 모터 캘리브레이션
			명령어: /mot_calib [part_id]
			기능 설명: 특정 관절 모터의 영점을 재설정합니다. (예: /mot_calib arm_l)
			예상되는 사이드이펙트: 캘리브레이션 순간(약 2초) 해당 관절의 토크(Torque)가 0이 됩니다. 들고 있는 물건이 있다면 즉시 바닥으로 떨어지므로 주의해야 합니다.

		7.2.4 네트워크 콜드 리셋
			명령어: /net_reset --cold
			기능 설명: 양자 통신 모듈을 물리적으로 껐다가 다시 켭니다. 클라우드 연결 오류(Error 210) 해결에 사용됩니다.
			예상되는 사이드이펙트: 홈 IoT 연결이 약 1분간 끊깁니다. 로봇을 통해 제어하던 조명이나 TV가 일시적으로 응답하지 않을 수 있습니다.

		7.2.5 감정 엔진 재부팅
			명령어: /emo_reboot
			기능 설명: 감정 엔진(Emotion Engine)을 재부팅합니다. 로봇이 조울증 증세(Error 606)를 보일 때 사용합니다.
			예상되는 사이드이펙트: 재부팅 직후 약 5분간 로봇이 '무표정' 상태가 되며, 사용자의 농담에 반응하지 않을 수 있습니다(Cold-Face Syndrome).

		7.2.6 커널 패닉 복구
			명령어: /kernel_panic_recover
			기능 설명: OS 커널이 멈춘 상태에서 안전 모드로 복구를 시도합니다.
			예상되는 사이드이펙트: 성공 확률은 80%입니다. 실패 시 로봇이 공장 초기화되며, 모든 사용자 데이터가 소실될 수 있습니다. (최후의 수단으로만 사용)
